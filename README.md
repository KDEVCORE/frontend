## Intro
Web 개발 목적의 실습을 위한 프로젝트입니다. Back-End에서 인증(Authentication), 권한 부여(Authorization) 구현이 목적입니다.
## Requirements
다음의 요구사항을 만족합니다.
- Membership
    - 회원 가입
    - 로그인
    - 회원 로그인 & SSO 로그인
- Contents
    - Open content
    - Main content (Member only)
